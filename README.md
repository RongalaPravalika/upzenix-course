A **responsive and elegant registration form** built with React, featuring form validation, show/hide password, PAN & Aadhaar validation, and a details page using React Router.

## **Features**

- **Form Fields**: First Name, Last Name, Username, Email, Password, Phone (with country code), Country, City, PAN, Aadhaar  
- **Validation**:
  - Required fields check  
  - Email format validation  
  - Password minimum 6 characters  
  - PAN: 5 letters + 4 digits + 1 letter  
  - Aadhaar: 12 digits  
- **Show/Hide Password** functionality  
- **Submit button** is disabled until all fields are valid  
- **React Router**: Redirect to a details page showing submitted data  
- **Responsive design** with elegant UI and gradient background  

---

## **Technologies Used**

- React 18+  
- React Router DOM  
- CSS for styling (modern and responsive)
